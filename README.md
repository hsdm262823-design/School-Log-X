## 📖 프로젝트 소개

**School Log X**는 학생이 학교 생활을 더 체계적으로 관리할 수 있도록 도와주는 웹 앱입니다.  
React 기반으로 제작되었으며, 개인 학습 및 학교 관련 정보를 한 곳에서 관리할 수 있습니다.

## 🚀 시작하기

### 사전 요구사항

- Node.js 18 이상
- npm 또는 yarn

### 설치 및 실행

```bash
# 저장소 클론
git clone https://github.com/hsdm262823-design/School-Log-X.git

# 프로젝트 폴더로 이동
cd School-Log-X

# 의존성 설치
npm install

# 개발 서버 실행
npm start
```
