# 04. 정보 구조 (Information Architecture)

## 1) 화면 목록
- 시작
- 홈
- 일정 상세
- 설정

## 2) 화면 이동 구조
시작 → 홈 → 일정 상세 → 홈  
홈 → 설정 → 홈

## 3) 메뉴 구조
| 메뉴 | 하위 메뉴 | 설명 |
|---|---|---|
| 일정 | 오늘 일정, 과목별 일정 | 과제/시험 일정을 확인하고 관리한다 |
| 등록 | 과제 등록, 시험 등록 | 새 과제/시험 일정을 추가한다 |
| 설정 | 계정, 알림 | 계정 기본정보와 알림 옵션을 관리한다 |

## 4) 권한별 접근
| 화면/기능 | MVP 단일 사용자(비로그인) |
|---|---|
| 시작 | 가능 |
| 홈 | 가능 |
| 일정 상세 | 가능 |
| 일정 등록 | 가능 |
| 설정 | 가능 |
