# 04. 정보 구조 (Information Architecture)

## 1) 화면 목록
- 시작
- 로그인/회원가입
- 홈
- 일정 상세
- 설정

## 2) 화면 이동 구조
시작 → 로그인/회원가입 → 홈 → 일정 상세 → 홈  
홈 → 설정 → 홈

## 3) 메뉴 구조
| 메뉴 | 하위 메뉴 | 설명 |
|---|---|---|
| 일정 | 오늘 일정, 과목별 일정 | 과제/시험 일정을 확인하고 관리한다 |
| 등록 | 과제 등록, 시험 등록 | 새 과제/시험 일정을 추가한다 |
| 설정 | 계정, 알림 | 계정 기본정보와 알림 옵션을 관리한다 |

## 4) 권한별 접근
| 화면/기능 | 비로그인 | 로그인 사용자 | 관리자 |
|---|---|---|---|
| 시작 | 가능 | 가능 | 가능 |
| 로그인/회원가입 | 가능 | 가능 | 가능 |
| 홈 | 불가 | 가능 | 가능 |
| 일정 상세 | 불가 | 가능 | 가능 |
| 일정 등록 | 불가 | 가능 | 가능 |
| 설정 | 불가 | 가능 | 가능 |
