# 08. 보안 체크리스트 (Security Checklist)

## 1) 접근 정책 (MVP)
- [ ] 비로그인 단일 사용자 MVP 정책 문서화
- [ ] 인증 의존 코드/설정 제거 여부 점검
- [ ] 관리자 전용 기능 미포함 여부 점검

## 2) 데이터 보호
- [ ] 민감 정보 암호화/마스킹
- [ ] 개인정보 최소 수집
- [ ] 사용자 식별정보(계정 정보) 미수집 정책 준수

## 3) 입력값 검증
- [ ] 서버 측 검증 적용 (MVP 필수)
- [ ] SQL Injection 방지
- [ ] XSS 방지

## 4) 비밀값 관리
- [ ] API 키를 코드에 하드코딩하지 않음
- [ ] 환경변수로 관리
- [ ] 권한 최소화
- [ ] Azure App Service 설정값(Application Settings)으로 비밀값 관리

## 5) 로그/모니터링
- [ ] 에러 로그 수집
- [ ] 민감정보 로그 노출 금지
- [ ] 이상 징후 알림 설정
- [ ] Azure Monitor 알림 규칙(5xx 증가, 응답시간 급증) 설정
